This site tracks the ECE 5760 final project: Bayesian-network structure learning with an FPGA-oriented search path.

Current work centers on:

- hill-climber baseline for comparison
- MCMC/order-sampler variant for hardware search
- deterministic preprocessing for parent-set and local-score tables
- fixed-point scoring experiments for hardware-friendly log-add behavior

Content is edited as Markdown in `_pages/`, `_posts/`, `_projects/`, and `_news/`.
