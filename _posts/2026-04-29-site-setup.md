This site now uses al-folio through Jekyll.
Project updates can be made with Markdown files in `_pages/`, `_posts/`, `_projects/`, and `_news/`.

The GitHub Actions workflow builds the static site and publishes the generated `_site/` artifact to GitHub Pages.
